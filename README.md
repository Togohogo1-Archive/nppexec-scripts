# NppExec Scripts
Scripts used in the [NppExec](https://github.com/d0vgan/nppexec) Notepad++ plugin

### Running a Script
Press F6 in Notepad++, select the `<temporary script>` dropdown option, and click OK. Alternatively, a script can be run with a Keyboard Shortcut.

### Binding a Script to a Keyboard Shortcut
1. Press F6 in Notepad++ and select the `<temporary script>` dropdown option
2. Click `Save...` and give the script a name
3. On Notepad++'s menu bar, go to `Plugins` -> `NppExec` -> `Advanced Options...`
4. In the `Associated script:` dropdown, select the name of the script saved and click `Add/Modify`, then click `OK`
5. After Notepad++ has been restarted, go to `Macro` -> `Modify Shortcut/Delete Macro...`
6. Select the `Plugin commands` tab and scroll down to the saved script (usually all the way down)
7. Double-click the script row and assign a shortcut, then click `OK`
8. Click `Close`
